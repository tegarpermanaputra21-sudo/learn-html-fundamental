1. Tujuan

  • Peserta didik mengetahui sintaks-sintaks dasar HTML.
  • Peserta didik mmapu menyelesaikan permasalahan sederhana menggunakan HTML.

2. Dasar Teori

HTML (Hyper Text Markup Language) merupakan bahasa markup untuk mesyusun halaman web yang dapat dibaca oleh browser. Browser seperti Google Chrome atau Firefox akan menerjemahkan kode HTML menjadi tampilan visual halaman web.

Beberapa Elemen HTML yang sering Digunakan:

Tag Fungsi
<h1> - <h6> Judul atau heading
<p> Paragraf teks
<a> Link atau tautan
<img> Menampilkan gambar
<br> Line break
<hr> Horizontal rule/Garis Horizontal
<b> Bold text/teks tebal
<i> Italic text/teks miring
